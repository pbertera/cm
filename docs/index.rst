Welcome to Configuration Manager's documentation!
=================================================

Configuration Manager è un sistema di configuration management per sistemi e servizi UNIX.
Configuration Manager offre:

* una CLI di gestione e configurazione 
* un CMDB (Configuration Management Database)
* comandi facilmente estendibili e configurabili
* diversi livelli di operatività
* ...
